# Student User Guide

## Overview
This guide helps students navigate enrollment, schedules, invoices, library loans, and transcripts.

## Audience & Permissions
- Audience: Students
- Access: Personal records and allowed actions per policy

## Prerequisites
- Active student account
- Enrollment status set by registrar

## Getting Started
1. Sign in to the Student Portal.
2. Review dashboard widgets for classes, invoices, and notifications.

![Student Dashboard](images/student/dashboard.png)

## Core Tasks

### View and Update Profile
1. Navigate to `Profile`.
2. Update contact details; save.
3. Review privacy and notification preferences.

### Class Registration
1. Open `Registration`.
2. Search available classes by term.
3. Add classes to schedule; resolve conflicts.
4. Submit for approval if required.

![Registration](images/student/registration.png)

### Invoices & Payments
1. Go to `Finance > Invoices`.
2. Review invoice details and due dates.
3. Pay online and download receipts.

![Invoices](images/student/invoices.png)

### View Reports & Transcripts
1. Open `Reports > Transcript`.
2. Select term; generate PDF.

### Library Loans
1. Navigate to `Library`.
2. Search catalog; place holds.
3. Review loans and due dates.

## Troubleshooting
- Cannot register: check prerequisites or enrollment status.
- Payment failed: retry or contact support with reference number.
- Transcript empty: select correct term.

## FAQs
- How to request a refund? Submit a ticket via `Support`.
- Can I change my schedule? Use `Registration > Manage Schedule` within allowed window.

## Security Notes
- Keep your account credentials secure; enable MFA if available.
