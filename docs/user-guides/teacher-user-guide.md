# Teacher User Guide

## Overview
This guide covers classroom operations: attendance, gradebook, assessments, communications, and performance reports.

## Audience & Permissions
- Audience: Teachers and academic staff
- Access: Class-level permissions; can manage assigned classes and students

## Prerequisites
- Active teacher account
- Classes assigned by registrar/admin

## Getting Started
1. Sign in and open `Teacher Portal`.
2. Review your class list and schedule.

![Teacher Dashboard](images/teacher/dashboard.png)

## Core Tasks

### Take Attendance
1. Navigate to `Classes > Attendance`.
2. Select class and date.
3. Mark Present/Absent/Late; add notes.
4. Save and sync.

![Attendance](images/teacher/attendance.png)

### Manage Gradebook
1. Go to `Classes > Gradebook`.
2. Create categories (Homework, Exams).
3. Add assignments with max scores and due dates.
4. Enter scores; auto-calculated totals appear.
5. Finalize grades and submit.

![Gradebook](images/teacher/gradebook.png)

### Create Assessments
1. Open `Assessments`.
2. Create quiz/exam with question types.
3. Set availability window and instructions.
4. Publish and notify students.

![Assessments](images/teacher/assessments.png)

### Lesson Plans & Resources
1. Go to `Resources > Lesson Plans`.
2. Upload materials; tag by subject and grade level.
3. Share with classes.

### Communications
1. Open `Messages`.
2. Send announcements to class or individuals.
3. Review read receipts.

### Reports
1. Navigate to `Reports > Class Performance`.
2. Filter by date range and class.
3. Export results.

## Troubleshooting
- Cannot see class: confirm assignment with registrar.
- Grades not calculating: check category weights and grading schema.
- Assessment not visible: verify publish status and time window.

## FAQs
- Can I import grades? Use `Import CSV` in Gradebook.
- How to curve scores? Adjust category weights or apply curve on assignment.

## Security Notes
- Do not share student grades outside authorized channels.
