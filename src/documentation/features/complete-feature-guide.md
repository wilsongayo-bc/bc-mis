# Complete Feature Guide

This comprehensive guide covers all features of the Benedict College Management Information System with detailed, role-based instructions for every menu and functionality.

## 📊 Dashboard

### Overview
The Dashboard is your central hub for accessing key information and performing quick actions. Content is customized based on your role.

### For All Roles
**Access**: Dashboard (Home icon in sidebar)

**Common Features**:
- Personal welcome message
- Quick action buttons
- Recent activity feed
- Notifications and alerts
- Role-specific widgets

### Role-Specific Dashboard Views

#### SUPERADMIN/ADMIN Dashboard
**Widgets**:
- Total Users by Role (pie chart)
- Student Statistics (enrolled, pre-registered, total)
- Financial Overview (payments, outstanding balances)
- Library Statistics (books, borrowed, overdue)
- Recent System Activities
- Quick Actions (Add User, Add Student, View Reports)

**How to Use**:
1. Review key metrics at a glance
2. Click on any widget for detailed view
3. Use quick action buttons for common tasks
4. Monitor recent activities for system oversight
5. Check notifications for important alerts

#### REGISTRAR Dashboard
**Widgets**:
- Enrollment Statistics (by grade level)
- Pending Enrollments (requiring action)
- Document Submission Status
- Academic Calendar Events
- Recent Enrollment Activities
- Quick Actions (New Enrollment, View Students, Manage Schedules)

**How to Use**:
1. Monitor pending enrollments daily
2. Track document submission progress
3. Check academic calendar for important dates
4. Process enrollments using quick actions
5. Review enrollment trends
