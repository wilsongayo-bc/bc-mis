# Quick Reference Guide

## 🚀 Common Actions by Role

### For Students
- **View Grades**: Dashboard → My Courses (planned)
- **Check Schedule**: Timetable
- **View Payments**: Payments → My Payments
- **Search Books**: Books → Search
- **View Borrowing**: Borrow Records

### For Teachers
- **View Classes**: Courses → My Courses
- **Check Schedule**: Schedules or Timetable
- **View Students**: Students (in your courses)
- **Search Books**: Books

### For Registrars
- **Process Enrollment**: Enrollments → Add New
- **Create Schedule**: Schedules → Add Schedule
- **Verify Documents**: Students → Documents tab
- **Generate Transcript**: PDF Generation

### For Librarians
- **Add Book**: Books → Add New Book
- **Process Borrow**: Borrow Records → New Borrow
- **Check Overdue**: Library Management → Overdue
- **View Statistics**: Library Management → Reports

### For Administrators
- **Add User**: Settings → User Management → Add User
- **Add Student**: Students → Add New Student
- **Reset User Passwords**: User Management → Reset Password (single) or Bulk Actions → Reset Passwords ([Admin Password Reset Guide](/documentation/reference/admin-password-reset))
- **Reset Password**: Login → Forgot Password? ([User Password Reset Guide](/documentation/reference/password-reset))
- **Verify Email**: Profile → Send verification email ([Email Verification Guide](/documentation/reference/email-verification))
- **View Reports**: Reports
- **System Settings**: Settings

## ⌨️ Keyboard Shortcuts

- **Ctrl/Cmd + K**: Quick search
- **Ctrl/Cmd + D**: Go to Dashboard
- **Ctrl/Cmd + H**: Open Documentation
- **Ctrl/Cmd + S**: Save form
- **Esc**: Close modal/dialog

## 🔍 Search Tips

- Use quotes for exact phrases: "John Smith"
- Filter by type: student, course, book
- Use date ranges in reports
- Combine multiple filters

## 📊 Status Indicators

- **🟢 Green**: Active, Available, Completed
- **🟡 Yellow**: Pending, In Progress
- **🔴 Red**: Overdue, Error, Critical
- **⚪ Gray**: Inactive, Unavailable

## 🔐 Security

### Password Requirements
- Minimum 8 characters
- At least 1 uppercase letter
- At least 1 lowercase letter
- At least 1 number
- At least 1 special character

### Email Verification (Link + 6-Digit Code) ([Email Verification Guide](/documentation/reference/email-verification))
1. Open **Profile** and look for the status near your email:
   - **Email not verified** or **Email verified**
2. If not verified, click **Send verification email**.
3. Check your inbox (and **Spam/Junk/Promotions**) for the verification email.
4. Open the email and do both:
   - Click the **Verify Email** link/button (opens the verification page)
   - Copy the **6-digit verification code**
5. On the **Verify Email** page, enter the 6-digit code and click **Verify**.
6. Return to **Profile** and confirm it shows **Email verified**.

Notes:
- You may need to wait ~60 seconds before you can resend.
- The verification code expires (typically within 30 minutes). If it expires, request a new one from **Profile**.
- Changing your email in **Edit Profile** requires verifying the new address.

### Session Management
- Auto-logout after 30 minutes of inactivity
- Click "Stay logged in" when prompted
- Always log out when finished
