# Welcome, Registrar! 📋

As a Registrar in the Benedict College Management Information System, you have specialized access to manage academic records, student enrollment, course scheduling, and document requirements. This guide will help you efficiently perform your essential duties.

## 🎯 Your Role Overview

**Access Level**: Academic Records Management (Level 3)  
**Key Responsibilities**: Student enrollment, course scheduling, academic records, document management, and transcript generation

### What You Can Access ✅

- **Dashboard**: Enrollment metrics and academic statistics
- **My Profile**: Personal information management
- **Students**: View student information and limited editing for enrollment purposes
- **Courses**: View courses and manage scheduling
- **Subjects**: View subject information
- **Enrollments**: Full enrollment management (primary responsibility)
- **Payments**: View payment status for enrollment verification
- **Documents**: Full management of requirements and categories
- **Schedules**: Create and manage course schedules
- **Timetable**: View and coordinate schedules
- **PDF Generation**: Generate academic documents and transcripts
- **Documentation**: Registrar-specific guides

### What You Cannot Access ❌

- User Management (cannot create/modify user accounts)
- Financial Management (cannot process payments, view-only access)
- Library Management (no library operations)
- System Settings (no configuration access)
- Full student profile editing (limited to enrollment-related data)
- Reports (limited to enrollment-related reports)


## 🚀 Getting Started

### Step 1: Access Your Dashboard
1. Log in with your registrar credentials
2. Navigate to the **Registrar Dashboard**
3. Review key metrics:
   - Total enrolled students
   - Pending enrollments
   - Pre-registered students
   - Document submission status
   - Upcoming academic calendar events

### Step 2: Familiarize Yourself with Enrollment Management
1. Navigate to **Enrollments**
2. Review current enrollment status
3. Check pending applications
4. Understand enrollment workflow
5. Review enrollment statistics by grade level

### Step 3: Review Document Requirements
1. Go to **Documents** → **Requirements**
2. Review required documents for each grade level
3. Check document categories
4. Understand document verification process

### Step 4: Explore Schedule Management
1. Navigate to **Schedules**
2. Review current course schedules
3. Check for scheduling conflicts
4. Understand schedule creation process

## 📚 Core Registrar Tasks

### Student Enrollment Management

#### Processing New Enrollments
1. Navigate to **Enrollments**
2. Click **"Add New Enrollment"**
3. Select student (or create new student first)
4. Choose course and section
5. Set enrollment date
6. Set status (Pre-registered → Verified → Enrolled)
7. Verify payment status (view only)
8. Click **"Save Enrollment"**

#### Verifying Student Documents
1. Go to **Students** → Select student
2. Click **"Documents"** tab
3. Review submitted documents
4. Check document completeness
5. Mark documents as verified
6. Request additional documents if needed
7. Update document status

#### Managing Enrollment Status
1. Navigate to **Enrollments**
2. Find enrollment record
3. Update status as needed:
   - Pre-registered → Verified (after document check)
   - Verified → Enrolled (after payment confirmation)
4. Add notes about status changes
5. Save changes

### Course Scheduling

#### Creating Class Schedules
1. Navigate to **Schedules**
2. Click **"Add Schedule"**
3. Select subject and course section
4. Assign teacher
5. Choose day of week
6. Set start and end time
7. Assign room/location
8. System checks for conflicts
9. Resolve any conflicts
10. Click **"Save Schedule"**

#### Managing Schedule Conflicts
- **Teacher Conflicts**: Same teacher, same time
- **Room Conflicts**: Same room, same time
- **Student Conflicts**: Students in section have overlapping classes

**Resolution Steps**:
1. Review conflict details
2. Adjust time, room, or teacher
3. Check alternative options
4. Save when resolved

### Document Management

#### Setting Up Requirements
1. Go to **Documents** → **Requirements**
2. Click **"Add Requirement"**
3. Enter document name and description
4. Select category
5. Choose applicable grade levels
6. Mark as required or optional
7. Save requirement

#### Organizing Categories
1. Navigate to **Documents** → **Categories**
2. Create logical groupings:
   - Personal Documents
   - Academic Records
   - Medical Documents
   - Financial Documents
3. Set category priorities
4. Save categories

### Transcript Generation

#### Generating Student Transcripts
1. Navigate to **PDF Generation**
2. Select **"Student Transcript"**
3. Choose student
4. Select academic year/term
5. Review transcript preview
6. Click **"Generate PDF"**
7. Download or print
8. Provide to student/guardian

## 💡 Pro Tips for Registrars

1. **Process enrollments daily** to avoid backlogs
2. **Verify documents promptly** for smooth enrollment
3. **Check schedules regularly** for conflicts
4. **Coordinate with finance** on payment verification
5. **Maintain accurate records** for compliance
6. **Use filters and search** to find students quickly
7. **Generate reports regularly** for enrollment tracking

## 📞 Getting Help

- **Enrollment Questions**: Contact administration
- **Technical Issues**: Contact IT support
- **Payment Verification**: Coordinate with finance office
- **Schedule Conflicts**: Consult with academic affairs
