# Welcome, Finance Staff! 💰

As a Finance user in the Benedict College Management Information System, you manage payment processing, financial tracking, and revenue reporting.

## 🎯 Your Role Overview

**Access Level**: Financial Operations Management (Level 3)

### What You Can Access ✅
- Dashboard with financial metrics
- My Profile
- Students (view for payment verification)
- Enrollments (view for billing purposes)
- Payments (full management)
- Reports (financial reports)
- PDF Generation (receipts, financial documents)
- Documentation

### What You Cannot Access ❌
- User Management
- Course/Subject Management
- Enrollment Management
- Library Management
- System Settings

## 📚 Core Finance Tasks

### Recording Payments
1. Navigate to **Payments**
2. Click **"Add Payment"**
3. Select student
4. Enter amount and payment method
5. Add reference number
6. Save payment (receipt auto-generated)

### Tracking Outstanding Balances
1. Go to **Reports** → **Financial Reports**
2. Select **"Outstanding Balances"**
3. View students with unpaid fees
4. Export for follow-up

### Generating Financial Reports
1. Navigate to **Reports**
2. Choose report type
3. Set date range
4. Generate and export
